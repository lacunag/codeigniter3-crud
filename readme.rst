# CRUD CODEIGNITER 3

## Description Project

This project is an example for creating a simple crud. The main child functions:

- add, modify and delete
- form fields validation
- send data by get
- send data by post
## Installation

- Create database
- Execute Migrations
- Configuration baseURL and database
